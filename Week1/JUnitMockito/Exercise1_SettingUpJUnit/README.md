# Exercise 1 - Setting Up JUnit

This exercise demonstrates how to create a simple Maven project with JUnit 5 and write basic unit tests for a calculator.

## Files

- `pom.xml` - Maven configuration with JUnit 5.
- `src/main/java/com/cognizant/Calculator.java` - Sample calculator implementation.
- `src/test/java/com/cognizant/CalculatorTest.java` - Basic JUnit tests.

# Spring Core and Maven

## Overview

This repository contains the solutions for the **Spring Core and Maven** exercises from the **Cognizant Digital Nurture 4.0** program. The exercises introduce the fundamentals of Spring Framework, Dependency Injection (DI), Inversion of Control (IoC), and Maven project configuration.

---

## Exercises Included

### Exercise 1: Configuring a Basic Spring Application

**Objective**

- Create a basic Spring application.
- Configure Spring using XML configuration.
- Load the Spring IoC container.
- Retrieve and use Spring-managed beans.

**Concepts Covered**

- Spring Core
- Spring IoC Container
- XML Configuration
- Bean Creation

---

### Exercise 2: Implementing Dependency Injection

**Objective**

- Understand Dependency Injection using Spring.
- Configure dependent beans.
- Inject dependencies using Spring XML configuration.

**Concepts Covered**

- Dependency Injection (DI)
- Inversion of Control (IoC)
- Constructor Injection
- Setter Injection
- Bean Wiring

---

### Exercise 4: Creating and Configuring a Maven Project

**Objective**

- Create a Maven project named **LibraryManagement**.
- Configure Maven dependencies for Spring Framework.
- Configure the Maven Compiler Plugin for Java 1.8.

**Spring Dependencies**

- Spring Context
- Spring AOP
- Spring Web MVC

**Concepts Covered**

- Maven Project Structure
- Dependency Management
- Maven Plugins
- Build Configuration

## Learning Outcomes

After completing these exercises, you will be able to:

- Create Maven-based Spring projects.
- Configure Spring applications using XML.
- Understand the Spring IoC Container.
- Implement Dependency Injection using Spring.
- Manage project dependencies with Maven.
- Configure Maven plugins for Java projects.

---

## How to Run

1. Clone the repository.
2. Open the required exercise in VS Code or Eclipse.
3. Build the project using Maven.
4. Run the main application class.
5. Verify the output in the console.
